# Welcome to using Python to analyze Silvaco I-V characteristics.

Tonyplot is a pain to use for multiple files, so why not use an easier way.

## Steps to utilize repo contents
1. Download zip file of selected folder
2. Unzip file and open contents
3. Use Silvaco simulation files (.in extension) to generate your own device simulations.
4. Run the python data analysis code in an IDE to get easy output data.

## Software required:
1. SIlvaco: Atlas and Athena modules
2. Python
3. Numpy
4. Matplotlib
5. IDE
