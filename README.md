# Welcome to using Python to analyze Silvaco I-V characteristics.

Tonyplot is a pain to use for multiple files, so why not use an easier way.

## Steps to utilize repo contents
1. Open the selected folder
2. Copy and paste the code .in file in Silvaco
3. Run Silvaco to generate your own device simulations.
4. After simulation completion, copy the Python code.
5. Paste the code into your chosen IDE.
4. Run the python data analysis code to get easy output data

## Software required:
1. Silvaco: Atlas and Athena modules
2. Python
3. Numpy
4. Matplotlib
5. IDE (like VSCode or Pycharm)
